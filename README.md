# Análise de Dados em R (repositório pessoal de estudo e prática)
![R](https://img.shields.io/badge/R-276DC3?logo=r&logoColor=white)

Este repositório é o meu espaço de estudo contínuo em R voltado para análise de dados em economia aplicada e políticas públicas!
A ideia aqui é registrar, passo a passo, tudo o que eu realmente uso e aprendo em R: comandos, truques, armadilhas, padrões de limpeza de dados, modelagem econométrica e interpretação de resultados.

Não é um “curso fechado”, nem um manual oficial de R.
É um projeto vivo, em constante atualização.

Sempre que eu:

- entender um conceito novo;
- resolver um problema prático de dados;
- gerar um conteúdo que eu gostaria de reaproveitar depois;  
- rodar um modelo econométrico e interpreto,

Irei documentar aqui – com código comentado e explicação em português claro

## Objetivo do repositório
A motivação para esse projeto são duas: **<u>um caderno de aprendizado continuo e permanente</u>** onde eu organizo minha própria compreensão e uma **referência rápida reutilizável** um lugar pra copiar/colar quando eu precisar repetir tarefas clássicas de pesquisa empírica.

Nada aqui é “a forma definitiva de fazer”. É literalmente o caminho que eu estou fazendo para sair do conhecimento superficial e chegar em proficiência prática usando R em problemas reais.


## Conteúdo: Estrutura dos tópicos
O conteúdo desse guia segue uma sequência lógica de aprendizado progressivo. 

#### **1. Fundamentos de R** (`fundamentos`)


Coisas essenciais da linguagem e do ecossistema tidyverse.

Aqui entram tópicos como:

tipos básicos de objeto (numeric, character, factor, logical, data.frame, tibble);

leitura de dados (readr, haven, readxl, etc.);

manipulação com dplyr (filter, mutate, select, arrange, group_by + summarise);

criação e recodificação de variáveis (dummies, faixas etárias, escolaridade etc.);

controle de fluxo e funções próprias (if, purrr::map, funções simples com function(...) {}).

Pergunta que essa parte responde:
“Como eu faço X no R?”

Objetivo: ser direto, quase “mini cheatsheet comentado”.





Cada aula aborda no mínimo um assunto em específico do R (a depender do assunto). Portanto, a não ser que haja domínio prévio dos temas eu aconselho a seguir essa estrutura, mas fique a vontade para utilizar da forma que quiser e explora-lo da maneira que achar ter mais sentido para si.
